- Кагуя Оцуцуки не выше Legendary.
- Naruto Baryon / Ichigo full power подняты до Mythic при наличии.

Компенсация:
Команда владельца: /compensate_patch14

Выдаёт один раз каждому игроку:
+2000 💎 Фисташек
+20 🐉 Драконита
+600 очков мультипасса

Защита от повторной выдачи есть: patch14_luffy_restore_2026_05

Как загрузить:
1. Распаковать архив.
2. Загружать содержимое архива в корень репозитория, без вложенной папки PATCH14_FULL_PROJECT.
3. Commit changes.
4. Render -> Manual Deploy -> Deploy latest commit.

После запуска проверить:
/start, Коллекция, Профиль, Награды / магазин, Мультипасс / Донат, Админка -> Все игроки, /storage, Путь Луфи, Бесплатный сундук.
