PATCH 8 — ULTRA POLISH / STABILITY / MONETIZATION

Главное:
- Добавлено 862 единых карточных изображения в media/cards/*.jpg.
- Мифические карты намеренно оставлены без изображений: для них владелец может положить GIF/MP4 вручную.
- Добавлен media/asset_manifest.csv: список всех карт и их медиа-статуса.
- Добавлен tools_generate_card_assets.py для повторной генерации безопасных карточных заглушек.

Код и стабильность:
- JSON теперь сохраняется атомарно через временный файл + os.replace.
- Для anime_battle_data.json создаётся .bak перед перезаписью.
- SQLite-сохранение переведено на timeout=30 и WAL-журнал.
- Ошибки сохранения/загрузки теперь попадают в bot_runtime.log вместо полного молчания.
- Исправлен random.seed() в рейде: теперь используется локальный random.Random, глобальный рандом бота не ломается.

Платежи Telegram Stars:
- pre_checkout теперь проверяет payload, user_id, валюту XTR и точную сумму.
- Добавлен processed_payments, чтобы один и тот же платёж не выдавал награду дважды.
- Premium multipass теперь активируется автоматически до 100 уровня сразу после оплаты.
- Покупки пишутся в purchases с payment_id, типом, кодом, суммой и временем.

Баланс:
- Гарант редкости теперь срабатывает только на те редкости, которые реально есть в конкретном сундуке.
- Бесплатный сундук больше не может форсить легендарку/мифика через общий pity, если эти редкости стоят 0 в шансах.

Онлайн PvP:
- Очередь онлайн-поиска получила TTL 5 минут.
- Добавлена кнопка отмены поиска онлайн-боя.
- Очередь чистит дубли, старые записи и офлайн-игроков.

Безопасность репозитория:
- В .gitignore добавлены anime_battle_data.json, owner_ids.txt, right_hand_ids.txt, WAL-файлы SQLite и временные JSON-файлы.

Важно:
- Интернет-фото персонажей не вставлялись. В архиве используются безопасные единые карточные баннеры без копирования чужих артов.
- Для реальных артов клади файлы вручную в media/cards/<card_id>.jpg или media/cards_watermarked/<card_id>.gif.
