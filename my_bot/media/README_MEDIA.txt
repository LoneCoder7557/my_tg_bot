Медиа для карт:
1) Возьми id карты из cards.json, например naruto_baryon.
2) Положи файл в папку media с таким же именем:
   media/naruto_baryon.gif
   media/naruto_baryon.jpg
   media/naruto_baryon.png
   media/naruto_baryon.mp4
3) Бот сам покажет файл при открытии карты.

Медиа для арен:
media/arenas/ruins.jpg
media/arenas/city.jpg
media/arenas/void.jpg
media/arenas/forest.jpg
media/arenas/desert.jpg
media/arenas/temple.jpg

Поддерживаются .gif, .mp4, .jpg, .jpeg, .png, .webp.
