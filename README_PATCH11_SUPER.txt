Что важно:
1. Заливать в GitHub надо содержимое этой папки, а не папку внутрь папки.
2. В корне GitHub должны лежать: bot.py, cards.json, requirements.txt, Procfile, .gitignore, promo_codes.json, media/.
3. В Render обязательно нужен Persistent Disk:
   Mount Path: /var/data
   Environment: DATA_DIR=/var/data
4. После деплоя проверь команду /storage.
5. Разовая компенсация игрокам: /compensate_progress_bug
   Команду запускать только владельцу один раз.
